Utilities for bean development.

Usage

<project ...>
...
  <dependency>
    <groupId>dev.orne</groupId>
    <artifactId>beans</artifactId>
    <version>0.6.0</version>
  </dependency>
...
</project>

Features

Provides utilities for implementation and management of POJO Java Beans.

  • Bean identities

    Abstraction mechanism for entity identities on beans, also known as “Primary ID” or “Primary Key”.

  • Bean references

    Annotation based mechanism to declare and validate bean based references to unique entity references.

  • Apache Commons BeanUtils converters

    Adds extra supported conversions to Apache Commons BeanUtils.

  • Experimental SPI based Jackson polymorphism

    Adds SPI based name based polymorphism to Jackson.
